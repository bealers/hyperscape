# Claude Code Rules

This directory contains framework and project-specific rules for Claude Code.

## Purpose
Rules in this directory are automatically loaded by Claude Code to provide context-aware assistance for the Hyperscape project.

## Structure
- Framework rules: Technology-specific guidelines (React, TypeScript, etc.)
- Project rules: Hyperscape-specific patterns and conventions
- Integration rules: How different systems interact

## Usage
Rules are automatically applied when working with relevant file types. No manual configuration needed.

## Related
- `.cursor/rules/` - Cursor IDE rules (development standards)
- `.claude/skills/` - Automatic context-aware capabilities
- `.claude/commands/` - Manual workflow shortcuts
